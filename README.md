Publiceren via GitHub Pages:
1. Zet alle bestanden in een GitHub repository
2. Settings > Pages
3. Deploy from a branch
4. Kies main en /root
5. Open de GitHub Pages-link

Installeren:
- Android: open de link in Chrome en kies 'Install app' of 'Toevoegen aan startscherm'
- iPhone/iPad: open de link in Safari en kies 'Zet op beginscherm'
